#### 程序功能描述
* 用户登录、登出、注册等基础功能
* 用户向旅游助手提出问题（类似chatgpt）并得到准确的回答
* 用户可以利用旅游助手制定详细的旅游计划
* 用户可以修改生成的旅游计划
* 用户可以通过旅游助手得知美食、门票等与旅游相关的信息
